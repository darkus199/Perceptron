# AI Perceptron Lab 🧠
Interaktívny webový simulátor jednoduchého neurónu (perceptrónu) postavený v Reacte.

### Funkcie:
- Vizualizácia architektúry neurónu v reálnom čase.
- Výpočet **MSE Loss** podľa vzorcov z Excelu.
- Automatický tréning pre logickú funkciu **AND**.
- Geometrický náhľad na lineárnu separáciu dát.

### Inštalácia:
1. `npm install`
2. `npm run dev`
